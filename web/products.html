FAQ

How do I uninstall FreeAI?

Chrome extension — chrome://extensions → Remove.

Claude Code terminal — freeai claude restore, then npm rm -g @freeai.fyi/terminal. Add rm -rf ~/.freeai to clear local state.

macOS app — drag FreeAI from Applications to the Trash.

Does FreeAI change my real claude command?

No — it adds a reversible shell alias and never edits your Claude install or settings. Undo with freeai claude restore.

How do my terminal credits reach my account?

Link by email during freeai claude setup (or freeai claude link). Run freeai claude doctor to confirm.

Where do I see my earnings?

Your portal at freeai.fyi/redeem — sign in with the email you linked.
