# Screenshots

Product screenshots — source images for landing pages, store listings, docs, and social/OG cards.

## Conventions

- **Format:** PNG (lossless, good for UI). Use JPEG only for large photographic shots.
- **Naming:** lowercase, hyphenated, descriptive — e.g. `terminal-redeem-flow.png`,
  `extension-popup-active.png`, `desktop-macos-home.png`.
- **Retina:** if a shot is 2× density, suffix `@2x` (e.g. `portal-dashboard@2x.png`).

Drop new screenshots straight into this folder. Mention the filename(s) when you
want me to use them.
